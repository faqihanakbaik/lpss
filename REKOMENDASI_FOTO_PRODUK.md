### Responsive Images

```html
<!-- Gunakan srcset untuk multiple resolutions -->
<img
  src="produk-small.jpg"
  srcset="
    produk-small.jpg   800w,
    produk-medium.jpg 1200w,
    produk-large.jpg  1920w
  "
  alt="Tabungan Tijarah iB"
/>
```

### Lazy Loading

```html
<!-- Tambahkan loading="lazy" untuk performance -->
<img src="produk.jpg" loading="lazy" alt="Produk" />
```
