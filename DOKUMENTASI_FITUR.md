**Rumus Perhitungan:**

```javascript
function calculateAngsuran() {
  const pokok = jumlahPembiayaan;
  const bulan = jangkaWaktu;
  const rate = marginRate / 100;

  const margin = pokok * rate * (bulan / 12);
  const total = pokok + margin;
  const angsuran = total / bulan;

  return angsuran;
}
```

**JavaScript:**

```javascript
function initAccordion() {
  document.querySelectorAll(".faq-question").forEach((question) => {
    question.addEventListener("click", function () {
      const faqItem = this.parentElement;
      const answer = faqItem.querySelector(".faq-answer");
      const isOpen = faqItem.classList.contains("active");

      // Close all
      document.querySelectorAll(".faq-item").forEach((item) => {
        item.classList.remove("active");
        item.querySelector(".faq-answer").style.maxHeight = "0";
      });

      // Open current
      if (!isOpen) {
        faqItem.classList.add("active");
        answer.style.maxHeight = answer.scrollHeight + "px";
      }
    });
  });
}
```

**Cara Ubah:**

## üîß TROUBLESHOOTING

### Masalah Umum:

1. **Produk tidak muncul**

   - Cek console browser (F12)
   - Pastikan file `*-products.js` ter-load
   - Cek `console.log()` di `script.js`

2. **Modal tidak buka**

   - Cek `initModal()` dipanggil di `script.js`
   - Cek CSS `.modal { display: none; }`
   - Cek event listener di card produk

3. **Kalkulator tidak hitung**

   - Cek function `calculateAngsuran()` di `calculator-dynamic.js`
   - Cek ID input sesuai dengan JS
   - Cek console untuk error

4. **FAQ tidak expand**

   - Cek `initAccordion()` dipanggil di `smooth-enhancements.js`
   - Cek CSS `.faq-answer { max-height: 0; }`
   - Pastikan tidak ada konflik dengan `initFAQ()` di `script.js`

5. **Arrow carousel tidak sebelahan**
   - Cek CSS `.carousel-btn.prev` dan `.carousel-btn.next`
   - Pastikan `top` sama untuk keduanya
   - Gunakan `!important` jika perlu
